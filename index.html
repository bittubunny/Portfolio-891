Sign Language Recognition

This project uses MediaPipe for hand landmark detection and TensorFlow/Keras to train a neural network for real-time hand sign recognition. With over 90% accuracy and <100ms latency, it ensures seamless interaction via a standard webcam.

Leveraging Machine Learning Techniques for enhanced Fraud Detection in Banking Systems

This project involves building a machine learning model to detect credit card fraud in an imbalanced dataset of 284,807 transactions. Using algorithms like Logistic Regression, LightGBM, XGBoost, and ANN, along with Bayesian Optimization, a Voting Classifier achieved over 99% accuracy. The model, deployed with Streamlit, provides real-time fraud detection for banking systems.
